# QBRawGuard · 原盘通知

智能拦截 qBittorrent 中的 BDMV / ISO / DVD 原盘格式种子，杜绝 Emby 无法播放的媒体污染。

## 核心能力

| 拦截路径 | 方式 | 速度 |
|---------|------|------|
| 事件驱动 | `DownloadAdded` 事件秒级响应 | 秒级 |
| 快速拦截 | 标题预检 + 文件正则匹配 | 分钟级 |
| 全量兜底 | 低频全扫描（默认关闭） | 小时级 |

## 命中后处理

- **停止模式**：暂停种子 + 打标签
- **删除模式**：删除下载任务 → 清理源文件 → 清理入库记录 → 清理媒体库文件（四件套联动）
- **延迟回扫**：自动清理拦截后异步产生的孤儿入库记录

## 识别范围

- Blu-ray / UHD Blu-ray / 3D Blu-ray 原盘（BDMV/CERTIFICATE）
- DVD / HD DVD 原盘（VIDEO_TS/HVDVD_TS）
- VCD / SVCD 原盘
- 光盘镜像（ISO/IMG/NRG/MDF/CUE/BIN 等）
- **不误判** WEB-DL 的 m2ts 流（HBO Max 等流媒体底层封装）

## 安装

1. 将本仓库添加为 MoviePilot 插件市场源
2. 在插件管理中安装「原盘通知」
3. 进入设置配置 QB 下载器和拦截规则

## 配置项

- **基本设置**：启停、拦截模式、间隔时间、命中动作、QB 下载器
- **通知与测试**：通知通道、报警图、模拟通知字段
- **高级规则**：自定义识别正则（默认规则已覆盖主流原盘格式）
